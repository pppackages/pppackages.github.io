mPLPrd

The package mPLPrd performs inference inference in the context of regression-discontinuity designs using the mPLP approach of Cavaliere et al. (2025). The software provides the user (bias-corrected) estimators of the ATE based on local polynomials, as well as confidence intervals, standard errors, and more. Download here.
