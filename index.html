Welcome

This website contains software implementations of prepivoting-based inference procedures.

mPLPrd implements the mPLP procedure by Cavaliere, Gonçalves, Nielsen, and Zanelli (2025) for efficient ATE in Regression-Discontinuity Designs.

mPLPnp implements the mPLP procedure by Cavaliere, Gonçalves, Nielsen, and Zanelli (2025) for efficient inference in nonparametric regression.
