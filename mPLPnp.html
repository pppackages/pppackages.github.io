mPLPrd

The package mPLPrd performs inference in the context of nonparametric regression using the mPLP approach of Cavaliere et al. (2025). The software provides the user (bias-corrected) estimators of nonlinear conditional expectations based on local polynomials, as well as confidence intervals, standard errors, and more.
